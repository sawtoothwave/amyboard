One-command AMYboard deploy:

python deploy_auto.py

Notes:
- Auto-detects the AMYboard USB serial port on macOS.
- Sends sketch.py over the MicroPython REPL in paste mode using short base64 chunks.
- Writes directly to /user/current/sketch.py to avoid the web editor's live knob-state splicing.
- Soft-resets the board after a successful upload.

Optional:
- python deploy_auto.py --port /dev/cu.usbmodem1101
- python deploy_auto.py --sketch sketch.py
- python deploy_auto.py --no-reset
